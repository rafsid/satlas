satlas.models.basemodel.BaseModel.set_chisquare_mapping¶

BaseModel.set_chisquare_mapping(mappingDict)[source]¶

Sets the prior mapping for the different parameters. This will affect chisquare fits.

Parameters:mappingDict (dictionary) – Dictionary containing the functions that give the prior for the given parameter value. Use the parameter names as keys.
