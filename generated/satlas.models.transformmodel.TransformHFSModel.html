satlas.models.transformmodel.TransformHFSModel¶

class satlas.models.transformmodel.TransformHFSModel(*args, **kwargs)[source]¶

Create an HFSModel that applies both a pre-processing transformation on the input data and a post-processing transformation on the output data. Mathematically:

Both transformations are initialized to the identical transformation for one input argument.
