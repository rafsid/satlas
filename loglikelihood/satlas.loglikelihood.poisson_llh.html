satlas.loglikelihood.poisson_llh¶

satlas.loglikelihood.poisson_llh(y, f, x)[source]¶

Returns the loglikelihood for a Poisson distribution. In this calculation, it is assumed that the parameters are true, and the loglikelihood that the data is drawn from the distribution established by the parameters is calculated.

Parameters:
  • y (array_like) – Data to which is being fitted.
  • l (array_like) – Result from the model.
Returns:

Array with loglikelihoods for the data.
Return type:

array_like
